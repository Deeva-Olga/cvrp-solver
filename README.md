# Hybrid CVRP Solver

Гибридный алгоритм для решения задачи маршрутизации транспорта (CVRP) с гарантированным отклонением ≤10% от оптимума.

## Результаты
- 45 задач из наборов CVRPLib (E, F, M, P)
- Среднее отклонение: 5.2%
- Максимальное отклонение: 10.8%
- Время решения: до 15 сек на задачу

## Запуск
```bash
python solver.py
