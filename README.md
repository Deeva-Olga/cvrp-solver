# Гибридный решатель задачи CVRP

Решение задачи маршрутизации транспорта с ограничениями грузоподъемности (CVRP) с использованием гибридного метаэвристического алгоритма.

## Результаты

| Набор | Задач | Среднее отклонение | Оценка |
|-------|-------|-------------------|--------|
| **E** | 13    | 5.9%              | 10     |
| **F** | 3     | 3.0%              | 10     |
| **M** | 5     | 5.1%              | 10     |
| **P** | 24    | 4.8%              | 10     |
| **Итого** | 45 | **5.2%**          | **10** |

- Все задачи решены с отклонением ≤ 10.8% от оптимального решения  
- Валидация проведена через официальные `.sol` файлы CVRPLib  
- Среднее время решения: 8.3 сек (макс. 15 сек на задачу)

##  Алгоритм

Гибридный подход:
1. **Начальное решение**: Алгоритм экономии (Clarke-Wright Savings)
2. **Интенсивный локальный поиск**:
   - 2-opt (оптимизация внутри маршрутов)
   - 2-opt* (межмаршрутный обмен)
   - Relocate (балансировка загрузки)
3. **Адаптивная стратегия**: перезапуск при застревании в локальном минимуме

Особенности реализации:
- Точное соответствие формату CVRPLib (округление расстояний до целых)
- Проверка грузоподъемности на каждом шаге
- Детерминированность результатов

## Зависимости

**Ключевые наблюдения:**
- Слабая корреляция размерности ↔ качество (+0.28) → стабильное качество даже для 200+ клиентов
- Время решения растёт как O(n²), но укладывается в лимит 15 сек
- Наилучшие результаты на наборах F и P (<5% отклонение)
